基于微博的内容识别和管控系统

前端采用HTML5+CSS3经典组合; 后端采用Flask

欢迎来到基于微博的内容识别和管控系统

本系统可通过Python爬虫自动爬取用户微博图片和文字，通过算法分析用户关键词而进行用户分类

本系统通过微博API实现自动发水军微博、水军评论的功能

本模板基于HTML5+CSS3的基础上进行设计制作,仅支持高版本浏览器,低版本浏览器请自行更换

内容包含:微博热搜、用户识别分析、图片检测、管控水军等功能。
